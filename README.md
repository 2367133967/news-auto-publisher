## 系统架构说明

本系统采用 **“Agent 语义理解 + 独立执行层”** 的双层解耦架构：

1. **语义层（OpenClaw Agent）**：负责接收自然语言新闻稿，提取结构化信息，并向用户确认。其操作范围被严格限制，**无权直接访问文件系统或执行系统命令**。
2. **执行层（Python 脚本 + Windows 任务计划）**：
   - 实际的文件写入由独立的 `自动发布.py` 脚本完成，确保修改的精准性。
   - Git 版本同步由 Windows 任务计划程序定时触发，实现了与用户交互的异步解耦，保证了系统响应的低延迟与高稳定性。

## 安全设计
- Agent 的 Prompt 明确禁止执行任何系统命令（如 `rm`、`del`、`format`），仅允许进行文本提取与确认。
- 实际修改仅针对 `news_list.json` 文件，且通过 Python 脚本的参数化输入防止了命令注入。
- Git 推送凭据独立存储于系统层，Agent 不可见。
